# Brenda Vega — Visual Systems Portfolio

Portfolio profesional desarrollado para presentar proyectos de diseño visual, UI/UX, sistemas editoriales y contenido digital.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- GitHub Pages

---

## Proyecto

Este portfolio fue diseñado como una experiencia editorial y visual enfocada en:

- sistemas visuales
- diseño de producto
- branding digital
- dashboards
- campañas
- contenido editorial
- case studies interactivos

Incluye proyectos como:

- Kairos Research
- Phronesis
- Fidex Capital
- Lytryum
- Exponencial Cripto Club
